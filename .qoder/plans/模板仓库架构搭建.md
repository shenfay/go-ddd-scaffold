# Go DDD Template Repository 搭建计划

## 项目概述
构建基于GitHub Template Repository模式的DDD企业级Go应用脚手架，采用标准的前后端分离目录结构，为开发者提供可直接使用的生产级架构模板。

## 核心目标
- 交付可直接fork使用的模板仓库
- 实现完整的DDD Clean Architecture分层结构
- 提供标准的开发环境配置
- 建立完善的文档体系

## 实施阶段

## 技术栈规范

### 后端技术选型
- **语言**：Go 1.21+
- **Web框架**：Gin
- **ORM**：GORM
- **数据库**：PostgreSQL 15+
- **缓存**：Redis 7+
- **日志**：Uber Zap
- **配置**：Viper
- **JWT**：golang-jwt/jwt

### 开发工具链
- **依赖管理**：Go Modules
- **测试框架**：testing + testify
- **代码质量**：golangci-lint
