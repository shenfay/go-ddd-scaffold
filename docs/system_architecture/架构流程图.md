## 🔄 核心业务流程

### 用户登录认证流程

```mermaid
flowchart TD
    A[📱 用户登录请求] --> B{🔍 验证码校验}
    B -->|通过| C[🔐 密码验证]
    B -->|失败| D[❌ 返回错误]
    C --> E{✅ 密码正确?}
    E -->|是| F[🎯 生成JWT Token]
    E -->|否| G[❌ 登录失败]
    F --> H[👤 用户信息打包]
    H --> I[📤 返回登录成功]
    G --> D
    I --> J[📱 客户端存储Token]

    style A fill:#4CAF50
    style D fill:#F44336
    style G fill:#F44336
    style I fill:#2196F3
```

## 🎮 交互场景流程

### 家长查看学习报告流程

```mermaid
sequenceDiagram
    participant P as 👩 家长APP
    participant API as 🌐 API服务
    participant RS as 📊 报告服务
    participant PS as 📈 进度服务
    participant DB as 🗄️ 数据库

    P->>API: 🔍 请求学习报告
    API->>RS: 📊 生成报告数据
    RS->>PS: 📈 获取学习进度
    PS->>DB: 🗄️ 查询学习记录
    DB-->>PS: 📤 进度数据
    PS-->>RS: 📥 统计信息
    RS->>DB: 🗄️ 查询成就数据
    DB-->>RS: 📤 成就信息
    RS-->>API: 📊 完整报告
    API-->>P: 📱 报告展示

    note over P,DB: 🎯 家长视角的数据整合流程
```

sequenceDiagram
