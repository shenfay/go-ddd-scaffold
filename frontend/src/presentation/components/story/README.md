# 数学剧情体验系统

## 项目概述

这是一个基于React的数学剧情学习体验系统，实现了毕达哥拉斯定理发现之旅的完整功能。系统采用渐进式增强策略，先实现简化但完整的功能验证核心概念，再逐步增强动画效果和交互体验。

## 核心功能

### 1. 时空穿越机制
- **触发按钮**：页面右下角的"时空穿越 🚀"按钮
- **流畅动画**：浮动、悬停和缩放效果
- **即时反馈**：用户交互时的视觉响应

### 2. 三幕剧情体验
- **现代教室**：学习直角三角形基础知识
- **古希腊集市**：与数学家毕达哥拉斯相遇
- **发现时刻**：验证并发现毕达哥拉斯定理

### 3. 丰富的视觉效果
- **粒子动画系统**：基于Canvas的动态粒子背景
- **场景切换动画**：平滑的颜色和布局过渡
- **角色出现动画**：毕达哥拉斯的弹出效果
- **数学公式动画**：定理展示的脉冲效果

### 4. 进度跟踪系统
- **可视化进度条**：显示当前学习进度
- **场景关联**：进度与场景切换同步更新
- **实时反馈**：进度变化的平滑过渡

### 5. 成就解锁机制
- **动态弹窗**：完成任务时的庆祝动画
- **成就展示**："数学探索者"成就
- **用户激励**：完成学习路径的正向反馈

## 技术特性

### 动画效果
- CSS3关键帧动画（浮动、弹跳、淡入淡出）
- React状态驱动的动态效果
- Canvas粒子系统
- SVG图形动画

### 交互设计
- 响应式按钮悬停效果
- 平滑的场景切换过渡
- 实时的视觉反馈
- 直观的操作指引

### 组件架构
- 模块化设计，易于扩展
- 状态管理清晰
- 代码结构简洁易懂

## 使用方式

1. 点击页面右下角的"时空穿越"按钮
2. 在三个场景间切换探索
3. 在"发现时刻"场景完成数学验证
4. 点击"完成探索"按钮获得成就
5. 点击"退出剧情"返回主界面
